PREREQUISITES:
  • Java Runtime Environment (JRE) 11 or higher
  • Windows 7 or later
  • ~50 MB free disk space

QUICK START:

  1. Extract Doable-Complete-Installer.zip to any folder
  2. Run: install.bat
  3. Follow the on-screen instructions
  4. The app will be installed to: C:\Users\[YourName]\AppData\Roaming\Doable
  5. A shortcut will be created in: Start Menu > Programs > Doable

RUNNING THE APPLICATION:

  After installation, you can run Doable from:
    • Windows Start Menu: Programs > Doable > Run Doable
    • Or double-click: Start Menu Shortcut
    • Or manually run: %APPDATA%\Doable\run-doable.bat

INSTALLATION TROUBLESHOOTING:

  ❌ Error: "Java is not installed"
    → Install Java from: https://adoptium.net/
    → Download the LTS version (Java 21 or 17 recommended)
    → Run the installer and restart your computer
    → Then run install.bat again

  ❌ Error: "JavaFX runtime components are missing"
    → Ensure you have Java 11+ installed
    → Reinstall Java from https://adoptium.net/
    → Make sure JavaFX is included with your JDK

  ❌ Error: "Command not found" or application won't start
    → Make sure all files from the ZIP were extracted
    → Verify the lib\ folder contains 12 JAR files
    → Try running: java -version in Command Prompt

UNINSTALLING:

  To remove Doable:
    1. Delete the folder: C:\Users\[YourName]\AppData\Roaming\Doable
    2. Right-click Start Menu shortcuts and delete them
    3. All data is stored locally - no registry entries to clean

FEATURES:

  ✓ Create, edit, and delete tasks
  ✓ Organize tasks by categories
  ✓ Set task priorities
  ✓ Custom repeat schedules (daily, weekly, specific days)
  ✓ Mark tasks as complete
  ✓ Desktop notifications for reminders
  ✓ Local database storage (SQLite)
