## üéØ Key Concepts

### The Feature
- **What**: Checkboxes for task reminders
- **Why**: Control reminders, auto-renew recurring tasks
- **How**: Click to mark/unmark, automatic renewal logic

### The Files Modified
- `Task.java` - Added `markedForCompletion` field
- `HomeController.java` - Added UI and logic

### The Functionality
- Suppress reminders for marked tasks
- Auto-renew due dates when unmarked
- Supports all repeat rule types

## ‚ú® Feature Highlights

From any documentation file, you'll learn about:

1. **Checkbox Display**
   - How tasks look with checkboxes
   - Visual indicator of state
   
2. **Toggle Behavior**
   - Marking tasks (suppress reminders)
   - Unmarking tasks (renew automatically)
   
3. **Renewal Logic**
   - Daily tasks advance 1 day
   - Weekly tasks advance 1 week
   - Custom intervals supported
   
4. **Reminder Suppression**
   - Marked tasks: no notifications
   - Unmarked tasks: reminders active
